## 문제 인식

1. 출력 방법에 관한 문제

## 문제 접근 

- System.out.println 
- System.out.print

## 코드

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String a = sc.next();
        
        System.out.println(a);
    }
}
```
