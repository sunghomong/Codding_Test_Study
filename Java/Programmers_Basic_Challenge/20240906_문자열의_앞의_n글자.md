## 문제 인식

1. my_string : 기준 문자열
2. n : 문자열을 잘라 줄 원소

## 문제 접근

- .substring(start,end) 를 통해 문자열 자르기

## 코드

```java
class Solution {
    public String solution(String my_string, int n) {
        return my_string.substring(0,n);
    }
}
```
